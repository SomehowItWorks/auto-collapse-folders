# Auto Collapse Folders (Obsidian Plugin)

⚠️这个插件的代码是由Google Gemini协助生成的。如果你很难接受生成式AI产出的程式，请务必谨慎下载。
⚠️The coding of this plugin is assisted by Google Gemini. If the idea of Ai-gen program concerns you a lot, please think twice before installing it.

顾名思义，这个简单的插件可以帮你自动折叠文件列表中不在活跃状态的目录。
As the name suggests, this simple plugin auto collapses the folders you aren't currently working on.
